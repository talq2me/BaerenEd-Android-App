Le Pirate Raté voyage sur la mer.
Son équipage voyage avec lui.


Le Pirate Raté regarde une carte au trésor.
Il regarde les points sur la carte.
Les points montrent la route vers le trésor.


Le bateau arrive à destination.
Le Pirate Raté aperçoit le trésor.
C'est une montagne de sucettes.


Le Pirate Raté demande de l'aide.
Son équipage et lui transportent le trésor.
Ils transportent l'énorme sac jusqu'au bateau.


Mais le trésor est très lourd.
Le bateau va couler! dit le canard.
- Qu'allons-nous faire? dit le singe.
-Qu'allons-nous faire? répète le perroquet.


Le Pirate Raté doit choisir...
son équipage ou les sucettes.


Le Pirate Raté reprend la mer.
Le singe, le canard et le perroquet l'accompagnent.


Le Pirate Raté a fait le bon choix.
Les vrais trésors sont les bons amis! dit-il.
